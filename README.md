# Patient Tool

## 使用說明

請先確保安裝以下工具/套件

- [node.js](https://nodejs.org/en/download)
- [pnpm](https://pnpm.io/installation)
- [mongoDB](https://www.mongodb.com/docs/manual/tutorial/install-mongodb-on-os-x/)

### 第一次安裝

```bash
cd patient-tool
pnpm install
```

### 運行專案

同時運行前後端:

```bash
pnpm dev:all
```

只運行前端:

```bash
pnpm dev:client
```

只運行後端:

```bash
pnpm dev:server
```
